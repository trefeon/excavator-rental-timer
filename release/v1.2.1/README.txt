===================================================
Excavator Rental Timer - Firmware Flashing Package
===================================================

Prerequisites:
1. Python installed on your PC (https://www.python.org/downloads/)
   * Make sure to check "Add Python to PATH" during installation.
2. CH340 or CP210x USB Drivers installed for your ESP boards.

Instructions:
1. Extract ALL files from this ZIP into a new folder on your PC.
2. Plug your ESP32 or ESP32-C3 board into the computer using a DATA USB cable.
3. Double-click the "flash.bat" file.
4. The tool will automatically install necessary dependencies.
5. Follow the on-screen numbers (1, 2, or 3) to flash the correct firmware to your board.
6. When prompted, you can choose to erase all data for a completely clean flash.
