🔌 Arsitektur Daya — 1 Batre untuk RC + ESP32

⚠️ Catatan Penting — 1 Batre untuk Semua

1. Buck converter harus mampu 3-5A (bukan 1A doang). Motor RC narik gede pas start/load.
2. Kapasitor 1000µF minimum di output buck — tanpa ini ESP32 reset tiap motor nyentak.
3. Voltage drop pas motor start: batre 7.4V bisa turun ke 5V. Buck converter harus survive di input drop.
4. ADC di ESP32 baca tegangan batre via voltage divider — kirim ke app buat monitoring.
5. Low voltage cut-off: kalo tegangan batre <6.0V (2.8V/cell LiPo), ESP32 pause timer + simpan state + kirim notifikasi ke app — operator ganti batre.

⚠️ Risiko + Mitigasi (1 Batre)

  • Motor RC narik arus → ESP32 reset → pake kapasitor 1000µF
  • Batre drop <6.0V → ADC detect → auto pause + notif ke app
  • Range BLE ~10m → staff HP harus dekat arena
  • Bucket converter rusak → ganti (stock spare 2 unit)
  • ESP32 boot cepat (~2 detik) dibanding ganti batre (5 detik)
  • Best practice: charging 10 slot + 2 unit cadangan total 12
